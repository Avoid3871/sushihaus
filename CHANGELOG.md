# Sushihaus Weiden — Changelog

## [1.0.0] — 2026-05-10

### Erstellt
- **Projekt-Setup**: React 19 + Vite + Framer Motion
- **Design System**: Japanisch inspiriertes dunkles Theme mit CSS Custom Properties
  - Farbpalette: Schwarz/Gold/Rot (墨/金/赤)
  - Fonts: Outfit (Display) + Inter (Body) via Google Fonts
  - Glassmorphism, Shadows, Spacing-System
- **SEO**: Schema.org Structured Data, Open Graph Meta-Tags, semantisches HTML
- **Navbar**: Sticky Glassmorphism-Navigation mit Scroll-Detection, Active-Section-Tracker, Mobile Hamburger-Menü
- **Hero Section**: Vollbild-Hintergrund mit Sushi-Foto, animierter Headline, Öffnungszeiten-Card, CTAs
- **Highlights**: 3-Spalten Feature-Grid mit staggered Scroll-Animationen
- **Speisekarte**: Interaktive Kategorie-Tabs (Vorspeisen, Maki, Inside-Out, Nigiri, Warme Gerichte, Desserts) mit animierten Einträgen und Dietary-Tags
- **Über Uns**: Restaurant-Interieur-Bild, Statistiken, Service-Badges, Beschreibungstext
- **Kontakt**: Glassmorphism Info-Cards, eingebettete Google Maps (Dark Mode), Service-Badges
- **Footer**: 4-Spalten Layout mit Navigation, Kontakt, Öffnungszeiten, Legal-Links
- **Responsive**: Mobile-first Design, alle Breakpoints (480px, 768px, 900px)
- **Animationen**: Framer Motion — Fade-in, Slide-up, Stagger, Spring-Tabs, Hover-Effects
- **Assets**: KI-generierte Food-Photography und Restaurant-Interieur-Bilder

### Briefing
Restaurant-Website für das Sushihaus in Weiden i.d. OPf. (Sebastianstraße 5). Japanisches Restaurant mit Sushi, Maki, Ramen, Bento-Boxen. Single-Page-App mit React + Vite, Framer Motion Animationen, dunklem Premium-Theme. Responsive für Desktop und Mobile. SEO-optimiert mit Schema.org Structured Data.
